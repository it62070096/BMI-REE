BMI โปรแกรมคำนวณ BMI, TDEE, REE
======

Detail

### ข้อดี / ข้อเสีย:

ข้อดี:

  * กรอกประวัติเพียงครั้งเดียว ได้ทราบถึง BMI, TDEE และ REE โดยที่ไม่ต้องกรอกข้อประวัติซ้ำ
  * สามารถวางแผนการควบคุมปริมาณที่จะรับเข้าสู่ร่างกายได้ โดยทำให้ได้รับพลังานส่วนเกินน้อยที่สุด

ข้อเสีย:

  * โปรแกรมยังคำนวณคาดเคลื่อนอยู่บ้าง
